NATIONAL PROJECT MONITOR

ABOUT

I’d like to welcome you to my web app ‘NATIONAL PROJECT MONITOR’. This web app was created for the purpose of finding a way of monitoring national projects using custom WEB GIS APPLICATIONS. It is a web app created as a demo for the purpose of the partial fulfilment of a degree in Geomatics Engineering from the Copperbelt University, Kitwe. The main objective is to develop a web-based GPS software program that Copperbelt University will use to track national projects. Accountability, transparency, and accessibility to different stakeholders will be improved as a result.

SUB-OBJECTIVES
  • Look into how GIS can be used to track national initiatives.
  • Create a GIS interface, data collection tool, and navigational application for the web to track national projects
  • Add information to the Web-based GIS platform for tracking and assessing national projects.
  • Showcase the system's capabilities for managing, monitoring, and disseminating reports for national projects.
